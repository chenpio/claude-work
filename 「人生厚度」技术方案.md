# 「人生厚度」日记小程序 — 技术方案

## 一、产品概述

情感向日记微信小程序，核心理念 **"记录 → AI复盘 → 回忆"**，帮助25-40岁职场人士沉淀生活、看见成长。

**核心 Slogan：** 记录人生的长度，沉淀生活的厚度

---

## 二、技术选型

| 层 | 选型 | 理由 |
|----|------|------|
| 前端框架 | **Taro 4.x + Vue 3** | 一套代码后续编译 H5/支付宝小程序 |
| UI 库 | **Taro UI** | 小程序原生组件，开箱即用 |
| 图表 | **ECharts 小程序版** | 情绪曲线、词云，仅复盘页引入 |
| 后端 | **微信云开发 CloudBase** | 免运维、内置鉴权、与小程序深度集成 |
| 数据库 | **云开发数据库**（文档型） | 灵活 Schema，适合日记多字段结构 |
| 文件存储 | **云开发存储** | 自带 CDN、图片压缩、权限控制 |
| AI | **DeepSeek API**（已有 Key） | 复盘生成、AI 追问 |
| 地图 | **腾讯地图 SDK** | 小程序原生兼容，城市级模糊定位 |
| 定时任务 | **云开发定时触发器** | 周日复盘、每日提醒 |
| 消息推送 | **微信订阅消息** | 日记提醒、复盘通知、沉默召回 |

## 六、核心云函数设计

### 6.1 AI 复盘 Prompt 结构

```
系统角色：你是一个温柔有共情力的生活复盘助手
输入：用户本周日记数组 [{date, content, mood, emotionTags, tags, oneLine, weather, location}]
输出：
  1. overview - 本周概览统计
  2. emotionCurve - 每天心情评分 + 关键词
  3. highlights - 1-3个高光时刻（附图）
  4. wordCloud - 高频关键词
  5. lowPoints - 低谷关注
  6. message - 温柔的鼓励寄语（150字内）
  7. suggestions - 1-2条具体下周建议

要求：温柔治愈，不生硬不鸡汤，贴合内容，不做无依据的猜测
```

### 6.2 数据安全措施

| 措施 | 实现方式 |
|------|---------|
| 传输加密 | HTTPS + 微信加密信道 |
| 存储加密 | 云开发自动加密 |
| 日记隔离 | `_openid` 天然隔离，用户只能访问自己的数据 |
| 位置模糊 | 前端 `getFuzzyLocation`，仅存城市级 |
| 图片权限 | 云存储设置仅创建者可读取 |
| 数据导出 | 云函数生成临时下载链接（1小时有效） |
| 图片脱敏 | 截图分享时前端自动覆盖隐私文字 |
| 彻底删除 | 删除账号时级联清除所有集合数据 |
